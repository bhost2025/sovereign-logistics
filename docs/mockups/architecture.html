Puntos clave de diseño:
  • Multi-tenant desde el día 1 — Row Level Security en Supabase garantiza aislamiento de datos por empresa.
  • Un contenedor puede tener productos de múltiples clientes finales (carga consolidada LCL). El modelo de datos lo soporta nativamente.
  • Next.js y Expo comparten tipos TypeScript — una sola fuente de verdad para los modelos de datos.
  • Todo marcado como "Fase 2" está contemplado en el esquema de BD desde el MVP para no romper la arquitectura al escalar.
